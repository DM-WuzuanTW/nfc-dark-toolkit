# ✍️ 寫入功能完整指南

## 🎯 支援的寫入類型（23 種）

### 📂 分類 1: 基本（3 種）

| 類型 | 圖示 | 說明 | 範例 |
|------|------|------|------|
| **純文字** | 📝 | 寫入任意文字內容 | "Hello World" |
| **網址** | 🌐 | 寫入網站連結 | "https://example.com" |
| **搜尋** | 🔍 | 寫入搜尋關鍵字 | "NFC 教學" → Google 搜尋 |

#### 使用範例：
```
1. 選擇「基本」→「純文字」
2. 輸入：「歡迎光臨」
3. 靠近標籤
4. ✓ 完成！掃描標籤會顯示文字
```

---

### 👥 分類 2: 社交網路（8 種）

| 平台 | 圖示 | 需要輸入 | 自動產生 |
|------|------|----------|----------|
| **Discord** | 💬 | 用戶 ID | `https://discord.com/users/{ID}` |
| **Instagram** | 📷 | 用戶名稱 | `https://www.instagram.com/{用戶名}` |
| **Facebook** | 👥 | 用戶名稱 | `https://www.facebook.com/{用戶名}` |
| **LINE** | 💚 | LINE ID | `https://line.me/ti/p/~{ID}` |
| **Telegram** | ✈️ | 用戶名稱 | `https://t.me/{用戶名}` |
| **Twitter/X** | 🐦 | 用戶名稱 | `https://twitter.com/{用戶名}` |
| **YouTube** | ▶️ | 頻道名稱 | `https://www.youtube.com/@{頻道名}` |
| **TikTok** | 🎵 | 用戶名稱 | `https://www.tiktok.com/@{用戶名}` |

#### 使用範例：
```
情境：想讓別人掃標籤後加你的 Instagram

1. 選擇「社交網路」→「📷 Instagram」
2. 只需輸入：「your_username」（不用輸入完整網址）
3. 預覽會顯示：https://www.instagram.com/your_username
4. 靠近標籤寫入
5. ✓ 完成！別人掃描後自動開啟你的 IG
```

### 📞 分類 3: 通訊（4 種）

| 類型 | 圖示 | 說明 | 自動處理 |
|------|------|------|----------|
| **Email** | ✉️ | 寫入 Email 地址 | 自動加上 `mailto:` |
| **聯絡人** | 👤 | 寫入聯絡資訊 | vCard 格式 |
| **電話** | 📞 | 寫入電話號碼 | 自動加上 `tel:` |
| **簡訊** | 💬 | 寫入預設簡訊 | 自動加上 `sms:` |

#### 使用範例：
```
情境：在名片上貼 NFC 標籤，掃描後直接撥號

1. 選擇「通訊」→「📞 電話號碼」
2. 輸入：「0912345678」
3. 預覽顯示：tel:0912345678
4. 寫入標籤
5. ✓ 掃描後自動詢問是否撥號
```

```

## 🌟 特色功能

### 1. 智能格式化
用戶**只需輸入關鍵資訊**，應用自動處理：

| 用戶輸入 | 自動處理為 |
|----------|-----------|
| `john_doe` (IG) | `https://www.instagram.com/john_doe` |
| `0912345678` (電話) | `tel:0912345678` |
| `25.033,121.565` (位置) | `geo:25.033,121.565` |
| `hello@mail.com` (Email) | `mailto:hello@mail.com` |

### 2. 即時預覽
輸入時**立即顯示**最終寫入內容

### 3. 分類清晰
**6 大分類** + **23 種類型**，井然有序

### 4. 中文界面
所有選項都是**繁體中文** + Emoji 圖示

```
